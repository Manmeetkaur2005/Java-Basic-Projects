# 💻 Java Basic Projects

This repository contains beginner-level Java projects to strengthen core programming concepts. Each project is built with simplicity, logic, and clear structure for better learning and understanding.

---

## 📁 Projects Included

| Project Name              | Description                                      |
|--------------------------|--------------------------------------------------|
| ✅ Student Grade Calculator | Calculates grades based on user input.          |
| ✅ ATM Simulator           | Simulates basic ATM operations (withdraw, check balance, etc.). |
| ✅ To-Do List (Console)    | Manages daily tasks using console input/output.  |
| ✅ Rock Paper Scissors     | Classic game built using Java logic.             |

---

## 🚀 How to Run

1. Clone the repository
2. Open any project folder in your preferred IDE (e.g., VS Code, IntelliJ, Eclipse)
3. Compile and run the `.java` file

```bash
javac FileName.java
java FileName
